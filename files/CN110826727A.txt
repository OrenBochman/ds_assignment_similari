Method and apparatus for optimizing machine learning models
The embodiment of the application discloses a method and a device for optimizing a machine learning model. One embodiment of the method comprises: constructing an objective function of a machine learning model; converging the target function to a local minimum point with a local flat neighborhood by utilizing a preset gradient-free learning algorithm; the machine learning model is optimized based on the local minimum points. The implementation mode provides a new gradient-free learning algorithm, and the target function of the machine learning model can be converged to a local minimum point with a local flat neighborhood, so that good generalization performance is obtained.
