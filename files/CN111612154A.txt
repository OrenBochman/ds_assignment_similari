Optimization method for Spark-oriented MLlib machine learning algorithm
The embodiment of the invention discloses an optimization method for a Spark-oriented MLlib machine learning algorithm. The method comprises the steps that step 1, acquiring a gamma matrix through extended matrix operation of a training data set, the gamma matrix comprises statistical information of the training data set, wherein the statistical information of the training data set is used for solving parameters of a machine learning algorithm; and step 2, solving parameters of a machine learning algorithm through the gamma matrix. Through the algorithm optimization method provided by the embodiment of the invention, the machine learning algorithm can be effectively optimized, and compared with the prior art, the space overhead and the time overhead are greatly reduced, so that the operation overhead of the system is reduced.
