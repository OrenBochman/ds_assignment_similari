Machine learning based handover parameter optimization
A method is disclosed, comprising: obtaining a plurality of handover parameter values; selecting a subset of switching parameter values from a plurality of switching parameter values using a first machine learning model; obtaining history information of a plurality of terminal devices; determining a first set of optimal handover parameter values for the plurality of terminal devices from the subset of handover parameter values; tagging the first set of optimal handover parameter values with historical information of the plurality of terminal devices to obtain a tagged data set; training a second machine learning model with the labeled data set, wherein the trained second machine learning model is capable of predicting a second set of optimal handover parameter values for the first terminal device based on historical information of the first terminal device.
