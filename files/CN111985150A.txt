Robust optimization design method for multi-layer electronic device by using machine learning to assist optimization
The invention discloses a robust optimization design method for a multi-layer electronic device by using machine learning to assist optimization. According to the method, a multi-layer electronic device robust optimization design architecture is constructed; and a machine learning auxiliary optimization method is introduced into each layer, so that existing information can be utilized to predict information including performance response of the electronic device, worst performance under specific input tolerance, maximum input tolerance overcapacity under specific output tolerance and the like,and the robust optimization design process of the electronic device is greatly accelerated. The method can be applied to the fields of robust optimization design and the like of electronic devices including antennas, arrays, active and passive microwave millimeter wave devices and the like.
